# Matrix Calculator Web

Браузерное приложение для вычисления определителя, произведения и обратной матрицы.

##  Идея и цель
Создать легковесный, отзывчивый калькулятор линейной алгебры, который:
- Позволяет вводить квадратные и прямоугольные матрицы (до 10×10 на старте)
- Вычисляет определитель, произведение и обратную матрицу с валидацией
- Демонстрирует чистое разделение логики и представления

## Функциональность
- [ ] Динамическая генерация сетки ввода (`<input type="number">`)
- [ ] Валидация: только числа, проверка размерности, запрет пустых ячеек
- [ ] `det(A)` → определитель (метод Гаусса, O(n³))
- [ ] `A × B` → произведение с проверкой совместимости `n == m`
- [ ] `inv(A)` → обратная матрица (Гаусс-Жордан с частичным выбором ведущего элемента)
- [ ] Обработка ошибок: неквадратная матрица, переполнение

##  Архитектура
- **Стек**: Vanilla JS (ES6+), HTML5, CSS3. Без сборщиков и фреймворков.
- **Принципы**: иммутабельность, разделение ответственности, прогрессивное улучшение.
- **Точность**: `Number.EPSILON` для сравнений с нулём, округление вывода до 4 знаков.
- **Хранение**: `localStorage` для сохранения последней сессии (опционально).

##  План развития
1. Инициализация проекта + документация (этот PR)
2. Ядро математики: `det()`, `multiply()`, `inverse()` + юнит-тесты
3. UI: сетка ввода, валидация, рендер результата, обработка ошибок
4. Интеграция: связка UI ↔ Core, `localStorage`, UX-полировка
5. Опционально: экспорт в CSV, поддержка комплексных чисел, тёмная тема
